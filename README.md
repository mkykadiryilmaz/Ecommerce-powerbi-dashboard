# Ecommerce-powerbi-dashboard
This project analyzes an e-commerce dataset to uncover insights related to sales performance, inventory management, and unsold products. The dashboard helps identify inefficiencies and supports data-driven decision making.

---

## Key KPIs

Average Sale Price: $59
Revenue: $10.73M
Unsold Items Cost: $9M
Products in Stock: 307.50K
Total Users: 100K
Average Age: 41.03

## Key Findings

Middle-aged customers are under-targeted → marketing campaigns should focus on this segment
Product variety for younger users can be expanded to increase engagement
Highest unsold costs are in categories like Jeans and Outerwear & Coats → discount strategies recommended
Some distribution centers have high stock but low sales → indicates inefficient inventory allocation
Asia has the highest user concentration → potential opportunity for regional expansion
High stock + high unsold trend suggests inventory planning inefficiencies
Product distribution between centers should be optimized based on regional demand
